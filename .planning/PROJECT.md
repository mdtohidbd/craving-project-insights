# Project: Craving - Fine Dining POS & Management System

## Context
Craving is a modern, high-end restaurant management system designed for the "Industrial Elegance" aesthetic. It features a robust POS, dynamic menu system, comprehensive returns management, and advanced financial analytics.

## Core Value
To provide restaurant owners with a premium, data-driven management experience that combines visual excellence with operational efficiency.

## Tech Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express (implied by directory structure)
- **Design System**: Industrial Elegance (HUD-style, premium UI)
